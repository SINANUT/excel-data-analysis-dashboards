# Excel Data Analysis Dashboards

This repository contains interactive dashboards built using **Microsoft Excel** to analyze business and financial data.  
The dashboards use Pivot Tables, Pivot Charts, slicers, and KPI metrics to transform raw data into meaningful insights.

## Projects Included

### 1. Retail Sales Dashboard
An interactive dashboard designed to analyze sales performance across beverage brands and retailers.

**Key Features**
- KPI cards showing **Total Sales, Units Sold, Price per Unit, and Operating Profit**
- Comparison of **Sales 2022 vs Sales 2023**
- **Variance analysis** to measure growth
- Sales analysis by **Beverage Brand** and **Retailer**
- Interactive **slicers for invoice date and brand filtering**

**Skills Demonstrated**
- Pivot Tables
- Data visualization
- KPI reporting
- Business performance analysis

---

### 2. Personal Finance Tracker
A dashboard built to monitor personal spending patterns and financial habits.

**Key Features**
- Expense tracking across categories such as **Living Expenses, Dining, Transport, Charity, and Medical**
- Monthly filtering to analyze spending trends
- Visualization using **Donut Charts, Waterfall Charts, and Treemap**
- Overview of **monthly income vs expenses**

**Skills Demonstrated**
- Financial data analysis
- Dashboard design
- Data storytelling
- Expense category analysis

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization Techniques

## Purpose of the Project
The goal of this project is to practice **data analysis, dashboard development, and visualization techniques** using Excel to extract insights from structured datasets.
